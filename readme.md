# Gemma's Kitchenette

A web-based Food Management and Ordering System built for **Gemma's Kitchenette**, a local carinderia in Tanza, Cavite.

The system helps manage daily food preparation, ingredient inventory, customer orders, and order processing through a centralized web application. 


## Features

### Customer
- Register and login
- Browse today's available menu
- Place food orders
- View order history
- Track order status

### Staff & Admin
- Manage food menu
- Manage ingredients and suppliers
- Record ingredient purchases
- Configure food recipes
- Prepare daily menu
- Automatically deduct ingredient inventory during food preparation
- Manage customer orders
- Update order status (Pending → Confirmed → Preparing → Out for Delivery → Completed)
